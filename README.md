# Restaurant Billing System

A responsive web-based Restaurant Billing System developed using HTML, CSS, and JavaScript. The application allows customers to select food items, adjust quantities, generate bills automatically, calculate GST, and print professional invoices.

## Features

* Customer Name and Phone Number Entry
* Interactive Food Menu
* Quantity Management using Plus/Minus Buttons
* Automatic Bill Generation
* GST Calculation
* Unique Bill Number Generation
* Date and Time Tracking
* Print-Friendly Invoice
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Select menu items and generate the bill.
